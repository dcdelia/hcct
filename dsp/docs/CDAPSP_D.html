Introduction

This component is an implementation of the fully dynamic all pairs shortest paths algorithm for distances up to D described in: V. King,"Fully dynamic algorithms for maintaining all-pairs shortest paths and transitive closure in digraphs", in Proceedings of the 40-th Annual IEEE Symposium on Foundations of Computer Science (FOCS'99), New York City, NY, pp. 81-99, 1999. Nodes pairs with distance higher than the maximum value D are reported to be at INFINITY.
