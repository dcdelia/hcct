Introduction

This component is an implementation of the fully dynamic all pairs shortest paths algorithm described in: V. King,"Fully dynamic algorithms for maintaining all-pairs shortest paths and transitive closure in digraphs", in Proceedings of the 40-th Annual IEEE Symposium on Foundations of Computer Science (FOCS'99), New York City, NY, pp. 81-99, 1999. Edges weights must be integers in the range [0,C).
