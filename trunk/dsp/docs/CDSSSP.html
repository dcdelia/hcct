Introduction

This component is an implementation of a simplified version of the fully dynamic single-source shortest path algorithm described in: G. Ramalingam and T. Reps, "An incremental algorithm for a generalization of the shortest path problem", Journal of Algorithms, 21:267-305, 1996. The implemented version is partially described in: C. Demetrescu, D. Frigioni, A. Marchetti-Spaccamela, and U. Nanni, "Maintaining shortest paths in digraphs with arbitrary arc weights: An experimental study", in Proceedings of the 4-th Workshop on Algorithm Engineering (WAE'00), Saarbruecken, Germany, September 2000.

CDSSSP_New

LGraph* inGraph
LGraph_TNode* inSource
LEdgeInfo* inEdgeCost

Creates a new CDSSSP object associated to inGraph. Note that this component does side effect on inEdgeCost when the user calls UpdateEdge. inGraph is modified too upon calls of InsertEdge or DeleteEdge. Caller is responsible of dellocating the created object using CDSSSP_Delete.
